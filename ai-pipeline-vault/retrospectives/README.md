# Retrospectivas

Reflexión por ciclo/sprint o al cerrar un milestone importante (ej. "migración a Optuna per-cluster", "ensemble top-3").

Formato: `Retro — [nombre del ciclo].md`. Usar template `tpl-retrospectiva`.

## Qué capturar

- Qué salió bien
- Qué podría mejorar
- 1-3 acciones concretas para el próximo ciclo
- Métricas informales (issues cerrados, PRs mergeados, deuda técnica creada vs pagada)
